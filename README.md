## 安裝

- 需要安裝 Node.js
- 需要安裝相關套件

```sh
npm install || yarn install
```

## 網站啟動

- 會啟動 sever 做測試頁面
- http://localhost:5000/
- port 可在 .env 修改

```sh
npm start
```

## 網站實際輸出

- 會壓縮 && 最小化程式碼

```sh
npm run build
```
